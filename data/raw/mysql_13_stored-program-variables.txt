13.6.4.1 局部变量 DECLARE 语句 13.6.4.2 局部变量作用域和解析
系统变量和用户定义变量可以在存储程序中使用，就像它们可以在存储程序上下文之外使用一样。此外，存储程序可用于 DECLARE 定义局部变量，存储例程（过程和函数）可声明为采用在例程与其调用者之间传递值的参数。
要声明局部变量，请使用 第 13.6.4.1 节“局部变量 DECLARE 语句” DECLARE 中所述的
语句。
变量可以直接用 SET 语句设置。请参阅 第 13.7.6.1 节，“变量赋值的 SET 语法” 。
